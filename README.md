# Adxparse

```bash
-d --directory   Parse Directory      parse -d /home/tux/mydir/ -c adxlib -o removespace
-f --file        Parse File           parse -f /home/tux/myfile.ads -c adxlib -o removespace
-r --rename      Rename Package       parse -r /home/tux/mydir/ -r adx-lib-sysout adx-lib-sysin
-l --list        Show Package List    parse -l /home/tux/mydir/
-m --metric      Show Metrics         parse -m /home/tux/myfile.gpr
-v --version     Show Version         parse -v
-h --help        Show Help            parse 
```
